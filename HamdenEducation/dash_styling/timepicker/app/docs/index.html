timepicker-ui

timepicker-ui is an easy library with timepicker. Fully wrote with pure JavaScript. This library is based on Material Design from Google.

These are examples of use timepicker-ui library. Fully wrote documentation you can find here or on the GitHub page.

This site is using Bootstrap 4, font Roboto and material-icons from Google.

Basic


HTML
            
            <div class="basic" >
              <input
                type="text"
                class="timepicker-ui-input"
                value="12:00 PM"
              />
            </div>
JavaScript
            
            import TimepickerUI from "timepicker-ui";
              
            const basicPicker = new TimepickerUI(basic);
            basicPicker.create();

Mobile version


HTML
            
            <div class="mobile" >
              <input
                type="text"
                class="timepicker-ui-input"
                value="12:00 PM"
              />
            </div>
JavaScript
            
            import TimepickerUI from "timepicker-ui";
              
            const mobile = document.querySelector(".mobile");
            const mobilePicker = new TimepickerUI(mobile, { mobile: true });
            mobilePicker.create();

Accept event

Timepicker value:


HTML
            
            <div class="accept-event" >
              <input
                type="text"
                class="timepicker-ui-input"
                value="12:00 PM"
              />
            </div>

            <p>Timepicker value: <span id="accept-value"></span></p>
JavaScript
            
              import TimepickerUI from "timepicker-ui";
              
              const acceptEvent = document.querySelector(".accept-event");
              const acceptValue = document.querySelector("#accept-value");
              const acceptEventInit = new TimepickerUI(acceptEvent, {
                enableSwitchIcon: true,
              });
              acceptEventInit.create();
              
              acceptEvent.addEventListener( "accept", ({ detail: { hour, minutes, type } }) => 
                  acceptValue.innerHTML = `${hour}:${minutes} ${type}`
              );
